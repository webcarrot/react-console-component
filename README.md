# react-console-component
Simple react component that display console messages

## INSTALL

```cmd
npm install react-console-component
```

## USAGE:

[Read source](/source/component/Console.js)

```javascript
// ...
import Console from "react-console-component";
// ...
<Console capturing />
// ...
```
